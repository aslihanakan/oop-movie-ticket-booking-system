This project is a simple **Movie Ticket Booking System** developed in Java.

## Project Description
The system allows users to:
- View available movies (2D and 3D)
- Select show times and seats
- Book tickets based on customer type
- Calculate ticket prices dynamically
- Run unit tests to verify system behavior

The project is designed using **Object-Oriented Programming principles** such as:
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
